# Kartu Statistik GitHub

Salin-tempel ini ke konten penurunan harga Anda, dan hanya itu. Sederhana!

Ubah `?username=` nilainya menjadi nama pengguna GitHub Anda.

```md
[![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra)](https://github.com/anuraghazra/github-readme-stats)
```

_Catatan: Rank yang tersedia adalah S+ (top 1%), S (top 25%), A++ (top 45%), A+ (top 60%), dan B+ (semuanya). Nilai dihitung dengan menggunakan [fungsi distribusi kumulatif](https://en.wikipedia.org/wiki/Cumulative_distribution_function) menggunakan komitmen, kontribusi, masalah, bintang, permintaan tarik, pengikut, dan repositori yang dimiliki. Implementasinya dapat diselidiki di [src/calculateRank.js](./src/calculateRank.js)._

### Menyembunyikan statistik individu

Untuk menyembunyikan statistik tertentu, Anda dapat meneruskan parameter kueri `&hide=` dengan nilai yang dipisahkan koma.

> Pilihan: `&hide=stars,commits,prs,issues,contribs`

```md
![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra&hide=contribs,prs)
```

### Menambahkan jumlah kontribusi pribadi ke jumlah total komitmen

Anda dapat menambahkan jumlah semua kontribusi pribadi Anda ke jumlah total komitmen dengan menggunakan parameter kueri `&count_private=true`.

_Catatan: Jika Anda menerapkan proyek ini sendiri, kontribusi pribadi akan dihitung secara default. Jika tidak, Anda harus memilih untuk membagikan jumlah kontribusi pribadi Anda._

> Pilihan: `&count_private=true`

```md
![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra&count_private=true)
```

### Menampilkan ikon

Untuk mengaktifkan ikon, Anda dapat meneruskan `show_icons=true` parameter kueri, seperti:

```md
![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra&show_icons=true)
```

### Tema

Dengan tema bawaan, Anda dapat menyesuaikan tampilan kartu tanpa melakukan [penyesuaian manual](#kustomisasi).

Gunakan `&theme=THEME_NAME` parameter seperti ini: -

```md
![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra&show_icons=true&theme=radical)
```

#### Semua tema bawaan: -

gelap, radikal, merko, gruvbox, tokyonight, onedark, kobalt, synthwave, kontras tinggi, dracula

<img src="https://res.cloudinary.com/anuraghazra/image/upload/v1595174536/grs-themes_l4ynja.png" alt="GitHub Readme Stats Themes" width="600px"/>

Anda dapat melihat pratinjau untuk [semua tema yang tersedia](./themes/README.md) atau memeriksa [file konfigurasi tema](./themes/index.js) & **Anda juga dapat menyumbangkan tema baru** jika Anda mau :D

### Kustomisasi

Anda dapat menyesuaikan tampilan Anda `Stats Card` atau `Repo Card` sesuka Anda dengan params URL.

#### Opsi Umum:

- `title_color` - Warna judul kartu _(warna hex)_
- `text_color` - Warna teks tubuh _(warna hex)_
- `icon_color` - Warna ikon jika tersedia _(warna hex)_
- `border_color` - Warna batas kartu _(warna hex)_. (Tidak berlaku saat `hide_border` diaktifkan)
- `bg_color` - Warna latar belakang kartu _(warna hex)_ **atau** gradien dalam bentuk  _sudut, awal, akhir_
- `hide_border` - Menyembunyikan batas kartu _(boolean)_
- `theme` - nama tema, pilih dari [semua tema yang tersedia](./themes/README.md)
- `cache_seconds` - atur header cache secara manual _(min: 7200, maks: 86400)_
- `locale` - atur bahasa di kartu _(e.g. cn, de, es, dll.)_
- `border_radius` - Pembulatan sudut pada kartu

> Catatan: Minimum cache_seconds saat ini adalah 4 jam sebagai perbaikan sementara untuk kelelahan PAT.

##### Gradien dalam bg_color

Anda dapat memberikan beberapa nilai yang dipisahkan koma dalam opsi bg_color untuk merender gradien, dengan format berikut:

```
&bg_color=DEG,COLOR1,COLOR2,COLOR3...COLOR10
```

> Catatan tentang cache: Kartu repo memiliki cache default 4 jam (14400 detik) jika jumlah garpu & jumlah bintang kurang dari 1k, jika tidak, 2 jam (7200 detik). Juga, perhatikan bahwa cache dijepit minimal 2 jam dan maksimal 24 jam.

#### Opsi Eksklusif Kartu Statistik:

- `hide` - Menyembunyikan [item yang ditentukan ](#menyembunyikan-statistik-individu) dari statistik _(nilai yang dipisahkan koma)_
- `hide_title` - _(boolean)_
- `card_width` - Atur lebar kartu secara manual _(angka)_
- `hide_rank` - _(boolean)_ menyembunyikan peringkat dan secara otomatis mengubah ukuran lebar kartu
- `show_icons` - _(boolean)_
- `include_all_commits` - Hitung total komit alih-alih hanya komit tahun ini _(boolean)_
- `count_private` - Hitung komit pribadi  _(boolean)_
- `line_height` - Mengatur tinggi garis antara teks _(angka)_
- `exclude_repo` - Kecualikan bintang dari repositori yang ditentukan _(nilai yang dipisahkan koma)_
- `custom_title` - Menetapkan judul khusus untuk kartu
- `text_bold` - Gunakan teks tebal _(boolean)_
- `disable_animations` - Menonaktifkan semua animasi dalam kartu _(boolean)_

#### Opsi Eksklusif Kartu Repo:

- `show_owner` - Tampilkan nama pemilik repo _(boolean)_

#### Opsi Eksklusif Kartu Bahasa:

- `hide` - Sembunyikan bahasa yang ditentukan dari kartu _(nilai yang dipisahkan koma)_
- `hide_title` - _(boolean)_
- `layout` - Beralih di antara dua tata letak `default` & `compact`
- `card_width` - Atur lebar kartu secara manual _(angka)_
- `langs_count` - Tampilkan lebih banyak bahasa di kartu, antara 1-10, default ke 5 _(angka)_
- `exclude_repo` - Kecualikan repositori yang ditentukan _(nilai yang dipisahkan koma)_
- `custom_title` - Menetapkan judul khusus untuk kartu

> :warning: **Penting:**
> Nama bahasa harus uri-escaped, sebagaimana ditentukan dalam [Persen Encoding](https://en.wikipedia.org/wiki/Percent-encoding)
> (yaitu: `c++` harus menjadi `c%2B%2B`, `jupyter notebook` harus menjadi `jupyter%20notebook`, dll.) Anda dapat menggunakan
> [urlencoder.org](https://www.urlencoder.org/) untuk membantu Anda melakukan ini secara otomatis.

#### Opsi Eksklusif Kartu Wakatime:

- `hide` - Sembunyikan bahasa yang ditentukan dari kartu _(nilai yang dipisahkan koma)_
- `hide_title` - _(boolean)_
- `line_height` - Mengatur tinggi garis antara teks _(angka)_
- `hide_progress` - Menyembunyikan bilah kemajuan dan persentase _(boolean)_
- `custom_title` - Menetapkan judul khusus untuk kartu
- `layout` - Beralih di antara dua tata letak `default` & `compact`
- `langs_count` - Batasi jumlah bahasa pada kartu, default untuk semua bahasa yang dilaporkan
- `api_domain` - Tetapkan domain API khusus untuk kartu, misalnya untuk menggunakan layanan seperti [Hakatime](https://github.com/mujx/hakatime) atau [Wakapi](https://github.com/muety/wakapi)
- `range` â€“ Request a range different from your WakaTime default, e.g. `last_7_days`. Lihat [dokumen WakaTime API](https://wakatime.com/developers#stats) untuk daftar opsi yang tersedia.

# Kartu Bahasa Teratas

Kartu bahasa teratas menunjukkan bahasa teratas yang paling sering digunakan pengguna GitHub.

_CATATAN: Bahasa Teratas tidak menunjukkan tingkat keahlian saya atau semacamnya; ini adalah metrik GitHub untuk menentukan bahasa mana yang memiliki kode paling banyak di GitHub. Ini adalah fitur baru dari github-readme-stats._

### Penggunaan

Salin-tempel kode ini ke readme Anda dan ubah tautannya.

Titik akhir: `api/top-langs?username=anuraghazra`

```md
[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra)](https://github.com/anuraghazra/github-readme-stats)
```

### Kecualikan repositori individu

Anda dapat menggunakan `&exclude_repo=repo1,repo2` parameter untuk mengecualikan repositori individual.

```md
[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&exclude_repo=github-readme-stats,anuraghazra.github.io)](https://github.com/anuraghazra/github-readme-stats)
```

### Sembunyikan bahasa individu

Anda dapat menggunakan `&hide=language1,language2` parameter untuk menyembunyikan bahasa individual.

```md
[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&hide=javascript,html)](https://github.com/anuraghazra/github-readme-stats)
```

### Tampilkan lebih banyak bahasa

Anda dapat menggunakan `&langs_count=` opsi untuk menambah atau mengurangi jumlah bahasa yang ditampilkan pada kartu. Nilai yang valid adalah bilangan bulat antara 1 dan 10 (inklusif), dan defaultnya adalah 5.

```md
[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&langs_count=8)](https://github.com/anuraghazra/github-readme-stats)
```

### Tata Letak Kartu Bahasa Ringkas

Anda dapat menggunakan `&layout=compact` opsi untuk mengubah desain kartu.

```md
[![Top Langs](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&layout=compact)](https://github.com/anuraghazra/github-readme-stats)
```

## Terapkan pada instance Vercel Anda sendiri

#### [Lihat Tutorial Video Langkah Demi Langkah Oleh @codeSTACKr](https://youtu.be/n6d4KHSKqGk?t=107)

Karena API GitHub hanya mengizinkan 5k permintaan per jam, saya `https://github-readme-stats.vercel.app/api` mungkin bisa mencapai pembatas kecepatan. Jika Anda meng-host-nya di server Vercel Anda sendiri, maka Anda tidak perlu khawatir tentang apa pun. Klik tombol sebarkan untuk memulai!

CATATAN: Sejak [#58](https://github.com/anuraghazra/github-readme-stats/pull/58) kami seharusnya dapat menangani lebih dari 5k permintaan dan tidak memiliki masalah dengan waktu henti :D
